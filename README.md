 Netflix Title Analytics
 
📌 Project Overview

Netflix Title Analytics is a data analysis project that explores Netflix's content library to uncover trends in movies and TV shows. Using Python, SQL, and Power BI, the project analyzes content distribution, release patterns, ratings, genres, and country-wise contributions to generate actionable insights from over 8,800 Netflix titles.

🎯 Objectives
Analyze Netflix's content catalog and identify key trends.
Understand the distribution of Movies vs TV Shows.
Discover top content-producing countries.
Analyze content ratings and audience categories.
Identify yearly content growth patterns.
Create interactive dashboards for business insights.
📊 Dataset Information
Dataset Size: 8,800+ Netflix Titles
Features: 12+ Columns
Content Types: Movies and TV Shows
Attributes: Title, Director, Cast, Country, Rating, Duration, Genre, Release Year, Date Added, etc.

📈 Key Insights
Netflix catalog contains 8,800+ titles.
Movies account for approximately 70% of the content library.
TV Shows represent around 30% of total content.
Significant growth in content production occurred after 2015.
The United States contributes the highest number of titles.
TV-MA and TV-14 are among the most common content ratings.
